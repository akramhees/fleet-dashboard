# Fleet Dashboard
# Full Stack Driver Tracking Platform

> This is a mini version demonstrating driver tracking, admin dashboards and campaign management

About this Project

This is a fullstack application that solves the core problems Wrapped Media is looking to tackle

- **Driver Management:** Mobile-friendly check-in/out system with GPS tracking
- **Admin Dashboard:** Live map view of all active drivers
- **Campaign Management:** CRUD operations for marketing campaigns
- **Reporting:** Analytics dashboard showing driver activity

---

Tech Stack used: 

| Layer | Technology |
|-------|------------|
| **Frontend** | React + Vite + Tailwind CSS |
| **Backend** | Node.js + Express |
| **Database** | PostgreSQL + Redis |
| **Maps** | Google Maps API |
| **Mobile** | PWA (Progressive Web App) |
| **Deployment** | Vercel (Frontend) + Render (Backend) |

---

Key Features

- ✅ Mobile-optimized PWA for drivers to start/end shifts
- ✅ Real-time GPS location tracking
- ✅ Live map showing all active drivers
- ✅ Full CRUD for marketing campaigns
- ✅ Offline GPS caching for low-connectivity areas
- ✅ Admin dashboard with analytics
