FleetPulse - Driver Tracking Platform

Live Demo: https://fleetpulse-app.onrender.com

About:

This is a mini fullstack driver tracking and campaign management platform. I built this using React for the frontend, Node.js and Express for the backend and PostgreSQL for the database. I integrated the Google Maps API for realtime GPS tracking and also implemented a full authentication system and deployed everything to Render with environment variables for securiy as well

Features
- Driver login with GPS tracking
- Google Maps integration
- Admin dashboard with campaign management
- PostgreSQL database
