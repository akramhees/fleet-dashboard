# 🚗 FleetPulse - Driver Tracking Platform

**Live Demo:** https://fleetpulse-app.onrender.com

## About

Full-stack driver tracking and campaign management platform built for Wrapped Media.

### Features
- Driver login with GPS tracking
- Google Maps integration
- Admin dashboard with campaign management
- PostgreSQL database
